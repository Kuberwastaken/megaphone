# Megaphone intro animation

A three-second, 1200×780 Remotion composition for the README demo. It preserves
the real Claude Code screen recording, easing in toward Megaphone when
dictation begins and pulling back as the finished text appears.

```bash
npm install
npm run start
npm run render
npm run render:gifs
```

The master MP4 is the best review format. The GIF script is provided for the
README and website exports. It uses FFmpeg's generated palette to keep the
animation crisp without making either page unnecessarily heavy.
