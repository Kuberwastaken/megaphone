# Megaphone intro animation

A 1176×764 source recording for the README demo across Claude Code, Codex,
Cursor CLI, and Google Docs. The checked-in MP4 is used directly; Remotion is
kept as a reproducible preview/render wrapper and adds no timeline edits.

```bash
npm install
npm run start
npm run render
npm run render:gifs
```

The master MP4 is the best review format. The GIF script is provided for the
README and website exports. It uses FFmpeg's generated palette to keep the
animation crisp without making either page unnecessarily heavy.
